FastTube: 유튜브 영상 스크립트 추출 웹 애플리케이션 개발

다음 요구사항에 맞는 유튜브 영상의 자막/스크립트를 추출하는 웹 애플리케이션 "FastTube"를 개발해주세요.
사용자는 유튜브 URL을 입력하면 해당 영상의 스크립트를 텍스트로 볼 수 있습니다.

## 기술 스택

- React + TypeScript (Vite 빌드 도구 사용)
- 상태 관리: Zustand
- API 통신: React Query + Axios
- UI 컴포넌트: shadcn/ui
- 스타일링: Tailwind CSS (Toss 디자인 시스템 참고)

## 핵심 기능

1. 유튜브 URL 입력 시 영상의 자막/스크립트 추출
2. 언어 선택 기능 (기본: 영어, 옵션: 한국어)
3. 추출된 텍스트 복사 및 다운로드 기능
4. 반응형 디자인 (모바일, 태블릿, 데스크탑)

## 자막 추출 핵심 코드

```
// ES6 / TypeScript
import { getSubtitles } from 'youtube-captions-scraper';
```

## UI/UX 요구사항

1. Toss 디자인 시스템 스타일 (깔끔한 여백, 미니멀 디자인)
2. 페이지 상단에 로고와 앱 이름
3. 중앙에 URL 입력 필드와 언어 선택 옵션
4. 결과 영역에 자막 텍스트와 기능 버튼 (복사, 다운로드)
5. 로딩 상태와 오류 메시지를 사용자 친화적으로 표시
6. 다크모드/라이트모드 지원

## 추가 고려사항

1. 자막 텍스트에서 검색 기능
2. 타임스탬프 표시 및 클릭 시 해당 시점으로 이동 기능
3. 영상과 스크립트를 나란히 볼 수 있는 레이아웃 옵션

### 주요 UI 구성 요소

1. **헤더 섹션**
   - 로고와 앱 이름으로 브랜딩 제공
   - 간결하면서도 인지도를 높일 수 있는 디자인
2. **메인 타이틀 영역**
   - 앱의 주요 기능을 명확하게 설명하는 타이틀
   - 간단한 설명 문구로 사용자에게 안내
3. **입력 컨트롤 섹션**
   - 언어 선택 버튼 (한국어/영어 토글)
   - YouTube URL 입력 필드
   - 변환 버튼 (로딩 상태 표시 기능 포함)
4. **결과 표시 영역**
   - 비디오 정보 (제목 및 채널명)
   - 임베디드 YouTube 플레이어
   - 자막 텍스트 영역 (스크롤 가능)
   - 복사 버튼 (복사 상태 표시 기능 포함)

### 주요 기능 설명

1. **언어 선택**
   - 한국어와 영어 중 원하는 자막 언어를 선택
   - 시각적으로 현재 선택된 언어를 강조 표시
2. **URL 입력 및 검증**
   - YouTube URL 입력 시 기본적인 유효성 검사
   - 잘못된 URL 형식에 대한 사용자 피드백 제공
3. **자막 추출 과정**
   - 로딩 상태 표시로 사용자에게 진행 상황 알림
   - 오류 발생 시 친절한 오류 메시지 제공
4. **결과 표시 및 관리**
   - 자막 텍스트 표시 (스크롤 가능한 영역으로 긴 텍스트 수용)
   - 원클릭으로 자막 복사 기능
   - 복사 완료 시 시각적 피드백 제공
5. **영상 재생**
   - 추출한 영상의 임베디드 플레이

### 해결 사항
1. 추출된 자막에서 리스트 형태로 보여줄 때, 새창에서 플레이 되는게 아니라 보고 있는 임베디드 된 영상이 재생되어야 한다.

2. 더보기 클릭 시 복사 다운로드 버튼이 생기는게 아니라, 더보기 토글 없이 바로 있어야 유저 접근성이 더 좋다. 수정하도록
